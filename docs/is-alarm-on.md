# is Alarm On

Get the alarm On/Off status

```sig
rtc.isAlarm()
```

## Return

* a [boolean](/types/boolean) to get the activation of the alarm. A ``True`` value means the alarm is **ON** and a ``False`` value means the alarm is **OFF**.
